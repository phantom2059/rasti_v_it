# AutoExam - Автоматическая проверка экзамена

Веб-сервис автоматической проверки экзамена по русскому языку для иностранных граждан.

## Технологии

- **React** - библиотека для создания пользовательского интерфейса
- **Vite** - быстрый сборщик и dev-сервер
- **TailwindCSS** - utility-first CSS фреймворк
- **Framer Motion** - библиотека для анимаций
- **React Router** - маршрутизация для SPA

## Установка

### Требования

- Node.js версии 14 или выше
- npm или yarn

### Шаги установки

1. **Установите все необходимые зависимости:**

```bash
npm install
```

Эта команда установит все необходимые пакеты, включенные в `package.json`:
- `react` и `react-dom` - основные библиотеки React
- `react-router-dom` - маршрутизация
- `framer-motion` - анимации
- `vite` - сборщик
- `tailwindcss`, `postcss`, `autoprefixer` - стилизация

## Запуск проекта

### Режим разработки

Запустите dev-сервер с hot-reload:

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:
- **Local:** http://localhost:5173/
- **Network:** http://[your-ip]:5173/

### Сборка для продакшена

Соберите оптимизированную версию для продакшена:

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`

### Просмотр продакшен-версии

Запустите локальный сервер для просмотра собранной версии:

```bash
npm run preview
```

## Работа с изображениями

Изображения в проекте импортируются через относительные пути. Чтобы добавить новое изображение:

1. Поместите файл в папку `images/` в корне проекта
2. Импортируйте его в нужном компоненте:
```jsx
import myImage from '../../images/myImage.png';
```
3. Используйте в JSX:
```jsx
<img src={myImage} alt="Описание" />
```

Vite автоматически оптимизирует изображения при сборке проекта.

## Решение проблем

### Порт уже занят

Если порт 5173 занят, Vite автоматически использует следующий доступный порт.

### Ошибки при установке

Попробуйте удалить `node_modules` и `package-lock.json`, затем установить заново:

```bash
rm -rf node_modules package-lock.json
npm install
```

### Проблемы с TailwindCSS

Убедитесь, что в `index.css` импортированы все директивы Tailwind:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### Изображения не отображаются

Убедитесь, что:
- Файлы изображений находятся в папке `images/`
- Импорт использует правильный относительный путь
- Расширение файла указано правильно (.png, .jpg, .jpeg и т.д.)
